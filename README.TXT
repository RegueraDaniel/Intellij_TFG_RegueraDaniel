ECONOFY es una aplicación de gestión de gastos y ahorros personal desarrollada con Spring Boot.

Descripción
Registro de gastos: Permite a los usuarios registrar y realizar un seguimiento de sus gastos diarios, clasificándolos por categorías (alimentación, transporte, entretenimiento, etc.).

Registro de ahorros: Los usuarios pueden registrar y realizar un seguimiento de sus ahorros mensuales, estableciendo metas y visualizando su progreso.

Informes y estadísticas: La aplicación genera informes y estadísticas detalladas sobre los gastos y ahorros de los usuarios, lo que les permite tener una visión clara de su situación financiera.

Recordatorios: Permite a los usuarios establecer recordatorios personalizados para fechas de pago, vencimientos de facturas o cualquier otro evento financiero importante.

Ejecución
Para ejecutar la aplicación, asegúrate de tener instalado [lista de requisitos previos].
A continuación, sigue estos pasos:

1. Clona el repositorio en tu máquina local o descarga el código fuente.
2. Abre el proyecto en IntelliJ IDEA (o tu IDE preferido).
3. Ubica el archivo TfgRegueraDanielApplication.java en la siguiente ubicación: com.iesvdm.tfg_regueradaniel.
4. Haz clic derecho en el archivo y selecciona "Run TfgRegueraDanielApplication" para iniciar la aplicación.
5. La aplicación se iniciará y estará disponible en [URL de tu aplicación] para que puedas acceder a ella en tu navegador web.

Dependencias
La aplicación utiliza varias dependencias de Spring Boot y otras bibliotecas. A continuación se enumeran algunas de las dependencias principales utilizadas en este proyecto:

spring-boot-starter-data-jpa: Proporciona soporte para la integración de JPA (Java Persistence API) en la aplicación.
spring-boot-starter-validation: Ofrece validación de datos y reglas de validación para la aplicación.
spring-boot-starter-web: Incluye las dependencias necesarias para construir una aplicación web.
spring-security-core: Proporciona características de seguridad para proteger la aplicación y sus recursos.
mysql-connector-j: Proporciona el controlador JDBC para conectarse a una base de datos MySQL.
lombok: Una biblioteca que ayuda a reducir el código repetitivo y simplifica el desarrollo.
